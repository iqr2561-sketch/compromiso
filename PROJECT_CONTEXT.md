# Contexto del Proyecto: Compromiso Diario

Este proyecto es una plataforma de diario digital moderna bajo el nombre **Compromiso Diario**, diseñada para ofrecer una experiencia de usuario rápida, visualmente atractiva y fácil de administrar.

## Vision
Transformar la información en una experiencia premium.
- **Carrusel de Noticias**: La sección principal rota automáticamente entre las noticias más relevantes.
- **Publicidad Integrada**: Espacios publicitarios estratégicamente ubicados sin comprometer la estética.
- **Acceso Histórico**: Botón para visualizar la versión anterior (clásica) del diario.
El objetivo es transformar un diseño estático en HTML a una aplicación React dinámica y escalable. Se prioriza el "wow factor" mediante el uso de:
- **Diseño Dark-First**: Estética premium con soporte para modo oscuro por defecto.
- **Jerarquía Visual Clara**: Uso de cuadrículas complejas (Hero, Sidebar, Flash News) para organizar la información.
- **Interactividad**: Micro-animaciones con Framer Motion y transiciones suaves.

## Características Principales
1. **Frontend Dinámico**: Las noticias se cargan desde un contexto global, permitiendo actualizaciones en tiempo real sin recargar la página.
2. **Panel de Administración**: Un sistema CRUD completo para agregar, editar y eliminar noticias, permitiendo gestionar qué noticias son destacadas (Hero) o rápidas (Flash).
3. **Responsive Design**: Optimizado para dispositivos móviles, tablets y escritorio.
4. **Secciones Categorizadas**: Organización automática por categorías (Tech, Actualidad, Deportes, Viral).
5. **Call to Action**: Sección de suscripción a Newsletter integrada con diseño de alto impacto.

## Estética
- **Colores**: Uso de una paleta vibrante sobre fondos oscuros profundos (`#101622`).
- **Tipografía**: Lexend, una fuente geométrica que mejora la legibilidad en pantallas.
- **Iconografía**: Lucide React para mayor coherencia visual y facilidad de desarrollo.
